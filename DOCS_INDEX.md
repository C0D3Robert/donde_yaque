## 🎯 Guías por Rol

### 👤 Usuario/Visitante

1. Lee: **README.md** (sección Características)
2. Sigue: **QUICKSTART.md** (para instalar localmente)
3. Explora: La aplicación en `http://localhost:5173`

---

### 👨‍💻 Desarrollador Backend

1. Lee: **README.md** (Stack Tecnológico + Estructura)
2. Sigue: **QUICKSTART.md** (Backend setup)
3. Estudia: **CONTRIBUTING.md** (Estándares Python + Testing)
4. Revisa: `backend/catalogo/models.py` y `views.py`
5. Consulta: **README.md** (API Endpoints)

---

### 🎨 Desarrollador Frontend

1. Lee: **README.md** (Stack Tecnológico + Estructura)
2. Sigue: **QUICKSTART.md** (Frontend setup)
3. Estudia: **CONTRIBUTING.md** (Estándares JS + React)
4. Revisa: **REFACTORIZATION_LOG.md** (Sistema CSS)
5. Modifica: `frontend/src/styles/variables.css` para temas

---

### 🎨 Diseñador/Personalización

1. Lee: **REFACTORIZATION_LOG.md** (entender variables)
2. Abre: `frontend/src/styles/variables.css`
3. Cambia: Colores, espacios, tipografía
4. Verifica: Los cambios en `http://localhost:5173`

---

### 🚀 DevOps/Deployment

1. Lee: **README.md** (Seguridad + Deployment)
2. Estudia: **CONTRIBUTING.md** (Deployment en producción)
3. Revisa: Configuración en `settings.py` (Backend)

## 📚 Archivos de Referencia

### Backend
- `backend/requirements.txt` - Dependencias Python
- `backend/core/settings.py` - Configuración Django
- `backend/catalogo/models.py` - Modelos de datos
- `backend/catalogo/views.py` - API Views
- `backend/catalogo/serializers.py` - JSON Serializers
- `backend/catalogo/admin.py` - Admin panel

### Frontend
- `frontend/package.json` - Dependencias Node
- `frontend/vite.config.js` - Configuración Vite
- `frontend/src/services/api.js` - Cliente HTTP
- `frontend/src/styles/variables.css` - Variables CSS
- `frontend/src/components/` - Componentes React

### Configuración
- `.env` - Variables de entorno (crear localmente)
- `.gitignore` - Archivos ignorados
